1. Abstract

Migration Reporting Tool este un utilitar bazat pe tehnologiile web. Gandit ca o solutie a unor procese ale vizualizarii, editarii și raportării datelor privind migrațiile, Migration Reporting Tool integrează observațiile utilizatorilor și datele provenite de la diverse senzori pentru a genera în timp real o hartă interactivă privind migrațiile efectuate de diferite entități.
